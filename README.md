Để chạy được ứng dụng cần download source code về và tiến hành khởi chạy cả front-end và back-end của hệ thống:

- Để chạy Frontend cần:
  ---> cd Client
  ---> npm install
  ---> npm run start
- Để chạy Backend cần:
  ---> cd Server
  ---> npm install
  ---> npm run start

Một số yêu cầu cần thiết:

- Cần cài Nodejs trên máy tính.
